CSS Units

Absolute Lengths (Fixed)

  1. px * pixels (1px = 1/96th of 1in)
  2. pc (1pc = 1/864 1 inch)
  3. Extras = cm,mm,inches,pi(picas)

Relative Lengths

r =root

e =element

m =multiply

v =viewport

IMP

  1. em = font-size of the element
    2. The font-size of the div element is set to 20px. The span element inside the div element has a font-size of 0.5em, which equals to 0.5x20 = 10px.
  2. rem = Relative to font-size of the root element
    3. The font-size of this div element is 1.8rem, which translates to 1.8 x the browser's font size.
    The font-size of this div element is 0.8rem. It also shows that it does not inherit from its parent element font size.
  3. vw = Relative to 1% of the width of the viewport*

    4. font-size: 12vw;

    Hello

    Resize the width of the browser window to see how the font-size of h1 changes.

    1vw = 1% of viewport width.

  4. vh = Relative to 1% of the height of the viewport*

    5. font-size: 12vh;

    Hello

    Resize the height of the browser window to see how the font-size of h1 changes.

    1vh = 1% of viewport height.

  5. vmin = Relative to 1% of viewport's* smaller dimension

    6. font-size: 12vmin;

    Hello

    Resize the browser window (both width and height) to see how the font-size of h1 changes.

    1vmin = 1vw or 1vh, whichever is smaller.

  6. vmax = Relative to 1% of viewport's* larger dimensio

    7. font-size: 12vmax;

    Hello

    Resize the browser window (both width and height) to see how the font-size of h1 changes.

    1vmax = 1vw or 1vh, whichever is larger.

    The vmax unit is not supported in Edge 15 and earlier, nor in Safari 6.1 and earlier.

  7. % = Relative to the parent element

    8. EXTRAS

  8. ex = Relative to the x-height of the current font (rarely used)
  9. ch = Relative to the width of the "0" (zero)
